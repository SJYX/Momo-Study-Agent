# Momo Study Agent 🧠

Momo Study Agent 是一个将“墨墨背单词”开放数据与 Gemini 深度大模型结合的自动化开源助手应用系统。它通过本地强数据拦截管理与 SSE 流驱动的高端可视化前端大屏，为你提供极度私有化、全自动的 AI 英语学习辅导。

## 🚀 核心特性

- 🤖 **多模态引擎直推**: 支持动态加载设定，接入极速 `gemini-3-flash` 级 AI 提供超高并发深层词根词缀幽默拆解。
- 🛡️ **本地隐私金库**: 所有待背数据及大模型创造的高价值私有财产全部由底层的 SQLite 持久化双重快照管控。不畏惧任何公有云端删库风险。
- 🌀 **SSE 无缝沉浸流**: FastAPI 中游利用流式管道直连苹果美学的单页应用前端（建设中）。
- 🔒 **全流程防污染**: 所有的开放验证 Token 和配置文件严格抽离受 `.gitignore` 保护。

## 🛠️ 安装清单 (Dependencies)
（该项目支持多维运行策略，如需配置可视化客户端需配置 Node.js 体系支撑）

- Python 3.12+ (uv 虚拟支持)
- FastAPI / Uvicorn (后方底座)
- SQLite3
- Google Generative AI (gemini sdk)

*注意：此版本代码为受权限保护剥离版，需自持配置本地 `config.py` 方可拉起。具体参见内建工程文档。*
