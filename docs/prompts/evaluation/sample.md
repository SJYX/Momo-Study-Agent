# 词汇提示词逆向评估样本 (Prompt Evaluation Sample)

> 评估目的：检查系统设定的 Prompt 是否让模型输出了过度简略或逻辑重复的废话。主要观察这四个极难发散的高频多义词表现。

**测试模型**: `mimo-v2-flash`
**测试词汇**: get, set, minute, run
---

## 【GET】

**📝 Basic Meanings (基本释义):**
v. 获得；取得；得到 (get a job; get a degree; get permission)
v. 理解；明白 (get the point; get an idea; get the message)
v. 抵达；到达 (get home; get to the airport; get there by 5)
v. 使；让 (get things done; get someone to help)

**🎯 IELTS Focus (雅思考点):**
- **核心考试逻辑**：'get' 是 IELTS 口语和写作中最常用的动词之一，但因其过于口语化，在正式写作中应优先使用更精确的同义词（如 obtain, achieve, arrive）。
- **测试模式**：常出现在口语 Part 1/2 的日常对话中，以及写作中作为‘使役动词’（get sb to do sth）的考查点。
- **技能上下文**：在阅读中，需注意 'get' 与不同介词搭配产生的不同含义（如 get over, get through）；在听力中，常用于日常指令或描述。

**🔗 Collocations (核心搭配):**
**get a grip**：控制情绪/局面（用于口语建议，如 'You need to get a grip!'）
**get rid of**：摆脱；处理掉（高频率写作话题，如环境、消费主义）
**get down to**：开始认真处理（用于描述学习或工作，如 'get down to work'）
**get across**：传达；使理解（用于沟通类话题，如 'get your message across'）

**⚠️ Traps (避坑陷阱):**
**Trap:** 形容词混淆：'get' 的过去分词 'got' 在英式英语中常用，但在美式英语或正式写作中，'have got' 表示‘拥有’时，应避免在 Academic Writing 中使用，优先用 'have'。

**🔄 Synonyms (同义替换):**
**get a job** → **secure a position** / **obtain employment**
**get better** → **improve** / **recover**
**get the message** → **grasp the implication** / **comprehend the point**

**⚖️ Discrimination (词义辨析):**
**get** vs **obtain**：**get** 侧重于日常、非正式的获取过程，常用于口语；而 **obtain** 侧重于通过正式、系统或努力的方式获得，更常用于学术或正式写作。

**📚 Example Sentences (场景例句):**
*[Writing Task 2 Context]: *To **get rid of** plastic waste, governments must implement stricter regulations and promote recycling.* [要摆脱塑料垃圾，政府必须实施更严格的法规并推广回收。]*
*[Speaking Context]: *I couldn't **get my point across** during the meeting because of the language barrier.* [由于语言障碍，我在会议上没能把我的观点讲清楚。]*

**🧠 Memory Aid (超级助记法):**
**记忆法一（核心逻辑）：** 'Get' 的核心是‘从无到有’或‘从A点到B点’的过程，无论是获得物品、理解信息还是到达地点。
**记忆法二（词根词缀/构词法）：** 源自古英语 'getan'，意为‘抓住、获得’，与 'gain' 同源，强调主动获取。
**记忆法三（场景/图像联想）：** 想象一个学生在考试中‘get’到一个好成绩（获得），然后‘get’到老师的表扬（理解），最后‘get’到大学录取通知书（抵达新阶段）。

**⭐ Word Ratings (单词价值评级):**
**提分杠杆率 (ROI): 9/10** - 口语和写作中极高频率出现，但需注意避免过度使用以提升语言正式度。
**学术输出潜力 (Academic Yield): 4/10** - 在学术写作中应谨慎使用，优先选择更精确的同义词。
**易错踩坑指数 (Trap Probability): 7/10** - 常因口语化过度或介词搭配错误而失分。

## 【SET】

**📝 Basic Meanings (基本释义):**
v. 放置；设置 (set a table; set a goal; set a record)
v. 规定；安排 (set rules; set a date; set the agenda)
n. 一套；一组 (a set of tools; a set of rules; a set of problems)

**🎯 IELTS Focus (雅思考点):**
- **核心考试逻辑**：在IELTS中，'set' 的高频考点在于其作为动词的‘设定目标/规则’及‘安排计划’的抽象含义，而非简单的‘放置’物理动作。
- **固定结构模式**：常考察 'set up' (建立), 'set out' (出发/阐述), 'set off' (引发/出发) 等短语动词的精确含义区别。
- **技能上下文**：在写作Task 2中，用于描述‘制定政策’(set policies) 或‘设定标准’(set standards)；在口语中，用于描述‘安排日程’(set a schedule)。

**🔗 Collocations (核心搭配):**
- **set a goal**：设定目标（写作中用于论证个人或社会发展的目标）
- **set up**：建立；设立（描述机构、系统或计划的建立，高频）
- **set out**：出发；阐述（用于描述旅程开始或详细说明观点）
- **set off**：引发；出发（用于描述触发事件或开始旅行）

**⚠️ Traps (避坑陷阱):**
**Trap:** 混淆短语动词 'set up' (建立) 与 'set off' (引发/出发)。例如，在描述‘引发争论’时，应用 'set off a debate'，而非 'set up a debate' (后者意为‘组织一场辩论’)。

**🔄 Synonyms (同义替换):**
**set a goal** → **establish an objective** / **define a target**
**set up** → **establish** / **institute**

**📚 Example Sentences (场景例句):**
*[Writing Task 2 Context]: Governments should **set** clear environmental **standards** to ensure sustainable development, which requires long-term commitment and public cooperation. [政府应设定明确的环境标准以确保可持续发展，这需要长期承诺和公众合作。]*
*[Speaking Context]: I usually **set** my alarm for 7 AM to **set** a routine for my morning study. [我通常把闹钟定在早上7点，为我的晨间学习设定一个规律。]*

**🧠 Memory Aid (超级助记法):**
**记忆法一（核心逻辑）：** 核心是‘确定位置或状态’，无论是物理放置（放桌子）、抽象规定（定规则）还是组成集合（一套工具），都围绕‘使某物处于特定状态’这一逻辑。
**记忆法二（词根词缀/构词法）：** 源自古英语 'settan'，意为‘使坐下’或‘放置’，与 'sit'（坐）同源，引申为‘使处于某种状态’。
**记忆法三（场景/图像联想）：** 想象一个项目经理在会议中 **set** a deadline（设定截止日期），同时在白板上 **set** up a chart（建立图表）来展示项目计划，这体现了从物理到抽象的‘设定’过程。

**⭐ Word Ratings (单词价值评级):**
- **提分杠杆率 (ROI): 9/10** - 作为超高频基础词，掌握其短语动词和抽象用法能直接提升写作和口语的准确性。
- **学术输出潜力 (Academic Yield): 8/10** - 在学术写作中频繁用于描述设定标准、建立系统或提出假设。
- **易错踩坑指数 (Trap Probability): 7/10** - 短语动词众多且含义易混，是IELTS考生的常见失分点。

## 【MINUTE】

**📝 Basic Meanings (基本释义):**
n. 分钟 (a few minutes; every minute; at the minute)
v. 记录；撰写会议纪要 (minute a meeting; minute the proceedings)
adj. 微小的；细致的 (minute details; minute differences; minute inspection)
adj. [生僻] 极小的；几乎看不见的 (minute particles; minute quantities)

**🎯 IELTS Focus (雅思考点):**
- **核心考试逻辑**：作为名词，'minute' 在听力和阅读中常与时间管理相关；作为形容词，在写作中用于描述细节或差异，是高分词汇。
- **固定搭配**：'at the minute'（此刻）在口语中常用；'minute details' 在写作Task 2中描述论证细节时非常有效。
- **技能应用**：在听力中注意数字和时间表达；在写作中，用 'minute' 修饰名词可提升描述精确性。

**🔗 Collocations (核心搭配):**
**at the minute**：此刻（用于口语或非正式写作，表示当前时间点）
**minute details**：微小细节（用于写作Task 2，强调论证的精确性）
**every minute**：每分钟；随时（用于描述频率或紧迫性）
**minute differences**：细微差别（用于比较类写作，提升语言精度）

**⚠️ Traps (避坑陷阱):**
**Trap:** 'minute' 作形容词时，拼写与名词 'minute' 相同，但发音不同（名词读 /ˈmɪnɪt/，形容词读 /maɪˈnjuːt/），在听力中易混淆；避免将形容词 'minute' 误用为动词 'minute' 的现在分词。

**🔄 Synonyms (同义替换):**
**a minute amount** → **a negligible amount** / **an infinitesimal amount**
**minute inspection** → **meticulous examination** / **scrupulous scrutiny**

**⚖️ Discrimination (词义辨析):**
**minute** vs **tiny**：**minute** 侧重于极小到几乎看不见的物理尺寸或细节，常用于学术或正式描述；而 **tiny** 更口语化，泛指任何小的东西，不强调精确性。

**📚 Example Sentences (场景例句):**
*[Writing Task 2 Context]: The **minute details** of the policy implementation must be scrutinized to ensure equity, as even **minute differences** in application can lead to significant disparities. [政策实施的微小细节必须被仔细审查以确保公平，因为应用中的细微差别可能导致重大差异]*
*[Speaking Context]: I have to leave in a **minute** because my meeting starts at the exact **minute** scheduled. [我得马上离开，因为我的会议将在预定时间准时开始]*

**🧠 Memory Aid (超级助记法):**
**记忆法一（核心逻辑）：** 'minute' 的核心是‘小’或‘时间单位’，名词表示时间分割，形容词表示极小，动词表示记录细节，所有含义都围绕‘精确细分’展开。
**记忆法二（词根词缀/构词法）：** 源自拉丁语 'minutus'（意为‘小的’），与 'minor'（较小的）同源，帮助记忆其‘微小’含义。
**记忆法三（场景/图像联想）：** 想象一个会议场景：你用笔‘记录’(minute) 每个‘分钟’(minute) 发生的‘微小’(minute) 变化，形成一个循环记忆链。

**⭐ Word Ratings (单词价值评级):**
- **提分杠杆率 (ROI): 8/10** - 该词在写作和口语中高频出现，且形容词用法能显著提升语言精度。
- **学术输出潜力 (Academic Yield): 9/10** - 在学术写作中描述细节和差异时非常有用，是高分词汇。
- **易错踩坑指数 (Trap Probability): 7/10** - 发音和词性转换易混淆，需注意上下文区分。

## 【RUN】

**📝 Basic Meanings (基本释义):**
v. 经营；开展；运行 (run a business; run a scheme; run an experiment)
v. 持续；延续 (run for three years; run until 2028; the contract runs)
n. 长期；连续阶段 (in the long run; a run of bad luck)
v. [生僻] 流经；延伸 (the river runs through; the road runs along)

**🎯 IELTS Focus (雅思考点):**
- **核心考试逻辑**：'run' 在 IELTS 中高频出现，尤其在 Writing Task 2 和 Reading 中，常用于描述系统、项目或政策的'运行/管理'，是'manage'或'operate'的高分替代词。
- **固定结构**：常见于抽象名词搭配，如 'run a business'（经营企业）、'run a campaign'（开展运动）、'in the long run'（从长远来看），这些是写作和口语的加分项。
- **技能侧重**：在 Reading 中常考其'持续'义（如事件持续时间）；在 Writing 中强调其'管理'义以提升词汇多样性。

**🔗 Collocations (核心搭配):**
- **run a business**：经营企业（写作中描述经济或商业话题）
- **run an experiment**：开展实验（学术写作中描述研究过程）
- **in the long run**：从长远来看（写作和口语中表达长期观点）
- **a run of bad luck**：一连串的厄运（阅读或口语中描述连续事件）

**⚠️ Traps (避坑陷阱):**
**Trap:** 混淆 'run' 作为动词的及物与不及物用法。例如，'run a business'（及物，管理） vs. 'the river runs'（不及物，流动）。在 IELTS 写作中，误用可能导致语法错误，如错误搭配介词（如 'run in a business' 而非 'run a business'）。

**🔄 Synonyms (同义替换):**
**Manage a project** → **Run a project** / **Oversee a project**
**Continue for a period** → **Run for a period** / **Last for a period**

**⚖️ Discrimination (词义辨析):**
**Run** vs **Manage**：**Run** 侧重于日常操作和持续性管理，常用于商业或项目语境，语气较口语化；而 **Manage** 侧重于战略规划和监督，更正式，适用于高层管理或复杂系统。

**📚 Example Sentences (场景例句):**
*[Writing Task 2 Context]: **In the long run**, investing in renewable energy can **run** a country's economy more sustainably than relying on fossil fuels. [从长远来看，投资可再生能源比依赖化石燃料更能可持续地运行国家经济。]*
*[Speaking Context]: I once helped **run** a small charity event, which taught me valuable organizational skills. [我曾协助组织一次小型慈善活动，这教会了我宝贵的组织技能。]*

**🧠 Memory Aid (超级助记法):**
**记忆法一（核心逻辑）：** 'Run' 的核心逻辑是'持续移动或管理'，从物理流动（河流流动）到抽象管理（经营企业），都围绕'持续进行'这一概念。
**记忆法二（词根词缀/构词法）：** 源自古英语 'rinnan'，意为'流动'，引申为'管理'，无特定词缀，但可通过'runner'（跑步者/管理者）联想。
**记忆法三（场景/图像联想）：** 想象一个CEO在河边跑步，同时思考如何'run'公司——这连接了'流动'和'管理'的双重意象，帮助记忆其多义性。

**⭐ Word Ratings (单词价值评级):**
- **提分杠杆率 (ROI): 9/10** - 'Run' 是 IELTS 高频词，掌握其多义和搭配能显著提升写作和口语的词汇多样性。
- **学术输出潜力 (Academic Yield): 8/10** - 在学术写作中，'run' 用于描述实验、项目或政策运行，具有高适用性。
- **易错踩坑指数 (Trap Probability): 6/10** - 其及物/不及物用法和固定搭配易混淆，但通过练习可有效避免。
