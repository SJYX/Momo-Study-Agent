# Web UI 工作区导航

本目录用于存放 Web 前端相关的执行计划与设计章节。

## 目录约定

- `chapters/`: 按日期与主题拆分的实施文档（计划、设计、评审记录）

## 维护规则

- 进行中的 Web UI 任务放在 `chapters/`。
- 明确已废弃或仅有历史意义的 Web UI 文档，统一归档到 `docs/history/web_ui_legacy/`。
- 本文件只做导航，不重复记录实现细节。

## 相关入口

- 项目开发导航: `docs/dev/DEVELOPMENT.md`
- 系统架构总览: `docs/architecture/ARCHITECTURE.md`
- 历史 Web UI 文档: `docs/history/web_ui_legacy/`
