# C07 Word/Sync/Preflight 二级页统一

状态：confirmed  
更新时间：2026-05-02

## 1. 章节目标

让二级页面在不增加心智负担的前提下与主流程语义一致，支持快速定位与处理。

## 2. 已确认决策

1. 统一程度：高统一
2. Word Library 默认入口：搜索框优先
3. Sync Status 默认视图：失败项优先
4. Preflight 默认结果排序：失败 > 警告 > 通过
5. Word 搜索交互：回车即查
6. Sync Status 默认展示最近一次 flush 结果摘要
7. Preflight 失败项提供一键复制修复建议
8. 二级页统一支持静音模式（仅红色告警）

## 3. 章节计划（可执行）

1. Word
   - 搜索框置顶
   - 回车即查
2. Sync
   - 失败项优先展示
   - 显示最近 flush 摘要
3. Preflight
   - 结果按失败/警告/通过排序
   - 失败项支持一键复制修复建议
4. 一致性
   - 三页统一静音模式
   - 状态词典与提示语义保持一致

## 4. 验收标准

1. Word 首页默认聚焦搜索并支持回车直查。
2. Sync 默认先展示失败项并可见最近 flush 摘要。
3. Preflight 默认先展示失败项，支持复制修复建议。
4. 三页均支持静音模式且仅保留红色告警。
5. 三页交互语义与主流程一致。

## 5. 风险与回退

1. 风险：高统一导致个别页特性被弱化。
   - 处置：保留页面特化字段，不改核心语义。
2. 风险：静音模式导致非关键信息遗漏。
   - 处置：提供显式“退出静音”入口并保留红色告警。
