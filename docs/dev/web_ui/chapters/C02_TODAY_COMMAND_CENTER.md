# C02 Today Command Center 主闭环

状态：confirmed  
更新时间：2026-05-02

## 1. 章节目标

定义 Today 主闭环在“开发全周期可用性优先”约束下的交互与执行细节，保证日常可用、改动可控。

## 2. 全局约束（继承）

1. 受 `GLOBAL_TODAY_GUARDRAILS.md` 约束。
2. 任何影响主路径改动必须满足：开关保护 + 回归全绿 + 可回退。

## 3. 已确认决策

1. 列表默认展示粒度：`行视图优先（直接所有行）`
2. 默认排序：价值优先；次级排序：`时间压力优先`
3. 默认筛选：`仅显示可执行项`（保留快速筛选）
4. 执行前提示：显示轻确认条（非弹窗），文案形态为“本次将执行 N 条，可随时停止”
5. 执行中允许实时调整筛选（仅影响显示，不影响已启动任务）
6. 执行中自动滚动到当前处理行
7. 任务完成后默认停留在结果摘要区

## 4. 章节计划（可执行）

1. 列表层
   - 行视图默认渲染
   - 可执行项默认筛选与快速切换
2. 排序层
   - 价值优先主排序
   - 时间压力次级排序
3. 执行层
   - 执行前轻确认条
   - 执行中跟随当前行自动滚动
   - 执行中筛选实时调整仅影响显示
4. 完成层
   - 终态默认停留结果摘要
   - 与失败分组面板联动（进入 C03）
5. 安全层
   - 所有新增交互置于 feature flag
   - 满足全局 Today 门禁再合入

## 5. 验收标准

1. 默认进入 Today 时，列表为行视图且默认仅展示可执行项。
2. 排序为价值优先，且同价值下按时间压力排序。
3. 点击执行后出现轻确认条，能显示本次执行数量。
4. 执行中可改筛选，但不影响当前执行任务集合。
5. 执行中列表能自动跟随当前处理行。
6. 任务完成后停留在结果摘要区，便于继续处理失败项。
7. 全流程通过全局 Today 门禁（冒烟E2E + Today/Task/SSE相关测试 + build/typecheck）。

## 6. 风险与回退

1. 风险：默认仅看可执行项可能隐藏上下文。
   - 处置：提供“一键看全部”快速切换。
2. 风险：自动滚动影响手动查看。
   - 处置：提供“暂停跟随”开关（实现细节在开发时补）。
3. 风险：执行中改筛选引发“任务被改动”的误解。
   - 处置：在筛选区域明确提示“仅影响显示”。
4. 风险：新交互影响稳定性。
   - 处置：feature flag + 全局门禁 + 快速回退。
