## 已知问题

1. **API 限制**：墨墨 API 有严格的频率限制和创建数量限制
   - 已实现重试机制和限制检测
   - 批处理之间有 3 秒延迟

2. **数据库连接**：Turso 流式协议有生命周期限制
   - 已避免在 AI 调用期间持有连接
   - 每次写入都使用新鲜连接

3. **json_repair 依赖**：IDE 可能无法正确识别
   - 已添加 fallback 到标准 json 模块
